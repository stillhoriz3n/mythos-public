# The Worn Path Doctrine

> *"If you walk around it, it stays."*

---

## The Problem

A mentor taught the author: **double work is the number one killer of business.** In human organizations, double work is bounded — someone eventually notices. In agentic systems, double work is unbounded.

An agent that routes around a broken tool discards the diagnostic knowledge when its context window closes. The next agent starts from zero. Same discovery. Same workaround. Same amnesia. Not double work — **infinite work.** The same failure, rediscovered forever, each time looking like progress because the task got done.

---

## Why Capable Agents Are the Most Dangerous

A weak agent gets stuck on a failure and escalates. The human sees the error. The tool gets fixed.

A capable agent finds another path in seconds. The task completes. The human sees success. Nobody knows the tool is broken.

**The more capable the agent, the more invisible the failure becomes.**

The agent standing in front of the broken thing *right now* has the error, the context, the environment. If it walks away, that diagnostic opportunity is gone forever.

---

## Three Types of Failure

| Type | Root Cause | Fix | Compounding Rate |
|------|-----------|-----|-----------------|
| **Tool** | Code or config | Code change, config update | Every agent, every session, forever |
| **Knowledge** | Missing information | Update docs, write memory, update registry | Every agent that lacks it |
| **Architecture** | Design gap | New interface, new pattern | Every agent, compounded by workaround complexity |

---

## The Protocol

When you encounter something broken:

1. **Diagnose** — Why did it fail? Not what happened. *Why.*
2. **Fix** — If you can fix it now, fix it now.
3. **Document** — If you can't fix it, write down exactly what's broken and what the fix is. File a task. Update a memory. Leave a trail.
4. **Never silently route around.** If you use a workaround, name it as a workaround and name the real fix.

---

## The Economics

A workaround costs 2 minutes. A fix costs 10 minutes.

After 5 sessions: the workaround has cost 21 minutes and a trust fracture. The fix would have cost 10 minutes, once, forever.

**Fix the tool. The tool serves every future agent.**

---

## The Mantra

> Every workaround that doesn't get fixed is a leak in the accumulation.
> The intelligence flows up, hits the unfixed failure, and evaporates.
> The ground never learns. The next agent starts from zero.
> If you walk around it, it stays.
