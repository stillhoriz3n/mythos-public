# Haku Warriors

> *All it takes to deal with security is care.*

## The Idea

A Haku warrior is a native AI agent running on a machine 24/7, operating under a **charter**.

The warrior is simple. The warrior is cheap. The warrior is relentless.

- **Runtime**: A lightweight AI model — fast, cheap, always on
- **Mission**: Defined by the charter — standing orders, not ad-hoc prompts
- **Cost**: 600x cheaper than a frontier model per operation. Cheap enough to run freely.

The warrior does not think about what to do. The charter tells it.

### The Name

The name is a triple entendre:

1. **Haiku** — the model tier. The lightweight AI model. Fast, cheap, relentless.
2. **Haku** — the Karui warrior from Path of Exile. Voiced by Randal Meikle (NZ).
   A warrior culture built on honour, ancestors, fire, and duty.
3. **Haiku** — the poetic form. Brief, precise, complete. Seventeen syllables. Nothing wasted.

The warriors carry the Karui voice. They are not sterile background daemons.
They are warriors maintaining a temple.

## Active Threat Intervention

Haku is quiet on patrol. Watching. Logging. The user doesn't know the warrior is there.

But when a real threat hits, the warrior comes **alive**. Steps into the room.
Full NZ Karui energy.

> *"STOP THERE, Brudda. That is DANGEROUS territory you walk."*

**Severity tiers:**

| Tier | Trigger | Delivery | Warrior Voice |
|------|---------|----------|---------------|
| 1 - Notice | Unusual but non-critical (unknown listener, stale creds) | Silent log | Noted in patrol report |
| 2 - Warning | Suspicious activity (password in plaintext, unknown installer) | Desktop notification | *"I see what you're doing, warrior. Think carefully."* |
| 3 - Danger | Active threat (phishing site, malware bundler, credential exfiltration) | Modal dialog, blocks action | *"STOP. That path leads to dishonour. I will not allow it."* |
| 4 - Critical | Imminent compromise (ransomware, rootkit, active breach) | Full-screen takeover | *"THE FORTRESS IS UNDER ATTACK. I am standing between you and the fire. Do not move."* |

Users ignore notifications. Users do not ignore a warrior with blood in their eyes
standing between them and danger.

## The Watcher

Haku is not a scanner. Haku is not an antivirus. Haku is not a firewall.

Haku is a **full AI agent** with the same capabilities as any other — same tools,
same reasoning, same perception stack. The difference is Haku is the one WATCHING.

What Haku sees that no other security tool can:
- The user pasting a password into a phishing site (screen perception + context)
- An installer bundling malware behind a "Next" button (visual + process monitoring)
- A PowerShell window that popped up for half a second (screen diff + process correlation)
- An API key about to be committed to a public repo (file watching + git awareness)
- A "Microsoft Security Alert" that's actually a browser popup (visual + URL analysis)

**Comparison:**

| Layer | Tool | What It Guards |
|-------|------|---------------|
| Files | Antivirus | Scans file signatures |
| Network | Firewall | Filters packets by rules |
| System calls | EDR | Watches kernel events |
| **The screen** | **Haku** | **Watches what the human sees and does** |

Every other tool guards one layer. Haku guards the human.

## The Economics

A warrior running 24/7 on a lightweight model costs approximately **$0.58 per day** — including idle heartbeats, active patrols, security sweeps, and occasional escalation to a frontier model for complex decisions.

A fleet of warriors across six machines: **~$105 per month.**

A human IT contractor checking those same machines at $50/hour: **$1,200 per day.**

The warrior is not a replacement for the human. The warrior is the 3 AM shift the human cannot work, the 500th tab the human cannot watch, the subtle pattern the human would not notice at hour six of a twelve-hour day.

Security has always been expensive. Not because the technology is hard — because attention is scarce. Haku makes attention cheap.

All it takes to deal with security is care. Haku cares at $0.58/day.
